## 如何运行项目

项目技术栈使用了：

- vue-cli@^4
- vue@^2
- tiptap@^1

首先 Clone 项目到本地：

```bash
git clone https://github.com/pftom/editure-vue.git
```

然后允许以下步骤开启项目：

```bash
cd editure-vue && npm install # 或者使用 yarn
npm run serve # yarn serve
```
